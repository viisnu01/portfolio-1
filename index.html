Smart India Hackathon 2024 ‚Äî Adaptive Modulation Recognition System

Developed a real-time adaptive modulation recognition system for DVB-S2X signals using Software Defined Radio (SDR) and deep learning, achieving over 95% classification accuracy.

This project was designed for the Smart India Hackathon 2024 (Software Edition) under the problem statement on enhancing wireless communication efficiency. Traditional systems use fixed modulation schemes, leading to inefficiency and increased error rates under changing signal conditions. Our proposed solution implements adaptive modulation and automatic modulation recognition (AMR) using deep learning integrated into a Software Defined Radio (SDR) testbed.

The system dynamically classifies modulation types ‚Äî BPSK, QPSK, 16-QAM, 64-QAM ‚Äî using real-time Signal-to-Noise Ratio (SNR) and Error Vector Magnitude (EVM) metrics. A custom GNU Radio algorithm with an embedded Python block enables on-the-fly waveform analysis and classification without requiring hardware-level reconfiguration.

The deep learning model combines CNN layers for feature extraction and LSTM networks for temporal sequence prediction, achieving a classification accuracy of 95.4% across 55 modulation schemes. Training data was generated using MATLAB‚Äôs Communication Toolbox for 1,10,000 waveform samples under various SNR and Eb/N‚ÇÄ conditions.

The model was deployed using the ONNX framework to ensure cross-platform compatibility and integrated with SDR hardware (HackRF, USRP, Xilinx RFSoC PYNQ) for real-time testing. Evaluation metrics such as Bit Error Rate (BER) and throughput confirmed the system‚Äôs robustness under varying channel conditions.

Tech Stack Used: GNU Radio, HackRF/USRP/Xilinx RFSoC PYNQ, MATLAB, Python, TensorFlow, ONNX, CNN-LSTM Architecture, DVB-S2X Signal Dataset.

ANAV ‚Äî ISRO Robotics Challenge 2025 - Autonomous Drone for GPS-Denied Environments

AI-powered quadrotor developed for GPS-denied, Mars-like environments ‚Äî selected among the Top 37 teams in the ISRO Robotics Challenge 2025 and advanced to the Elimination Round.

The Autonomous Navigation Aerial Vehicle (ANAV) was designed by Team XENITH from Kumaraguru College of Technology for the ISRO Robotics Challenge (URSC 2025). Built for autonomous operation in GPS-denied Martian conditions, the system integrates the NVIDIA Jetson Orin Nano and Pixhawk 2.4.8 flight controller to perform real-time sensor fusion, terrain mapping, and safe-spot landing.

Equipped with the MYNT EYE 3D Stereo Camera and SLAMTEC RPLiDAR S1, ANAV uses RTAB-Map SLAM, ICP, and Dijkstra‚Äôs Algorithm for precise localization and obstacle avoidance. A deep-learning module combining PointNet++ and ResNet detects safe landing zones, while an Emergency Response System ensures stability under power or link loss.

The drone achieved stable autonomous take-off, 30-second hover, and accurate landing during the qualification and elimination rounds, demonstrating advanced autonomy under Martian-like, GPS-denied conditions.

Tech Stack Used: Jetson Orin Nano, Pixhawk 2.4.8 (PX4), MYNT EYE 3D, RPLiDAR S1, ROS2, OpenCV, Python, RTAB-Map, Gazebo SITL, PointNet++, ResNet.

A Data-Driven Optimization Framework for Solar Power Generation and Storage Capacity Enhancement

Developed a data-driven optimization framework integrating machine learning and multi-objective optimization to enhance solar power generation and battery storage performance.

This project, completed during my 7th Semester Final Year at Kumaraguru College of Technology, focuses on improving solar power efficiency and reliability through data analytics, machine learning-based forecasting, and intelligent optimization algorithms.

The system forecasts solar irradiance and load demand using LSTM models and dynamically optimizes solar and battery parameters via Genetic Algorithm (GA) and Particle Swarm Optimization (PSO). Simulations were conducted in MATLAB/Simulink and Python to evaluate energy flow, battery utilization, and economic performance.

Validation employed real-world datasets from the Harvard Dataverse and optimization analysis using HOMER Pro and NREL System Advisor Model (SAM).

Tech Stack Used: MATLAB, Python, HOMER Pro, NREL SAM, NumPy, LSTM.

Outcome: Enhanced solar generation efficiency, extended battery life, and established a scalable, sustainable framework for data-driven solar optimization in microgrid applications.

Wire Color Sequence Detection System

Developed a real-time wire color and sequence detection system using NVIDIA Jetson Orin Nano and Arducam IMX477, designed as a low-cost alternative to industrial sensors like Keyence LR-W70C.

This project, carried out during my internship at Pricol Limited (Corporate Manufacturing Engineering Department), aimed to create a cost-effective vision-based color inspection system for wire harness verification. The system detects individual wire colors and validates their sequence to ensure compliance with product specifications during assembly.

The solution utilizes the Jetson Orin Nano Super Developer Kit paired with an Arducam IMX477 camera. Using OpenCV and HSV color space filtering, the system isolates wire colors in real-time, applies contour detection, and classifies each color accurately.

A PyQt5 GUI displays the live feed, detected colors, and sequence validation (‚ÄúCorrect‚Äù or ‚ÄúIncorrect‚Äù) in an intuitive interface. Detection logic is implemented through HSV thresholding and left-to-right positional sorting of identified colors.

The initial prototype was built using Raspberry Pi 5 with the Arducam IMX298 camera, later upgraded to Jetson Orin Nano for industrial-grade performance and GPU-accelerated processing. Future enhancements include multi-camera integration, AI-based classification for lighting variation handling, and PLC/SCADA connectivity for factory automation.

Tech Stack Used: Jetson Orin Nano, Arducam IMX477, OpenCV, Python, PyQt5, NumPy, HSV Color Model, GUI-based Vision System Design.

Smart Glove-Based Gesture-Controlled Robot

A wireless gesture-controlled Mecanum-wheeled robot using an ESP32 and MPU6050 sensor, enabling intuitive hand gesture navigation with real-time motion tracking and Wi-Fi communication.

The Smart Glove-Based Gesture Control System was designed to replace traditional joystick or app-based control with a more natural, hands-free approach. The system uses an ESP32 microcontroller and an MPU6050 inertial measurement unit (IMU) to detect hand movements and transmit corresponding commands to the robot wirelessly over Wi-Fi.

The glove detects motion through accelerometer and gyroscope data from the MPU6050, processed by the ESP32 transmitter module. Commands such as forward, backward, left, right, and stop are sent to another ESP32 module on the robot, which drives the motors via an L298N motor driver.

The robot features Mecanum wheels that allow omnidirectional movement, offering precise motion control in all directions ‚Äî forward, backward, diagonal, and rotational ‚Äî without changing orientation. Real-time control is achieved through I¬≤C communication between the ESP32 and IMU, ensuring low latency and smooth motion tracking.

The system‚Äôs flexible design makes it suitable for applications in assistive robotics, industrial automation, and search & rescue operations. Future enhancements include AI-based gesture prediction, computer vision integration, and autonomous path planning.

Tech Stack Used: ESP32, MPU6050, L298N Motor Driver, Mecanum Wheels, Python, Arduino IDE, I¬≤C Protocol, Wi-Fi Communication.

Pi Spy Rover ‚Äî Mecanum-Wheeled Surveillance Robot

A WiFi-controlled Mecanum-wheeled robot designed for 360¬∞ omnidirectional movement and live camera surveillance, developed using Raspberry Pi, ESP8266, and L298N motor driver modules.

The Pi Spy Rover is a Mecanum-wheeled robot built to demonstrate real-time omnidirectional mobility and remote surveillance. Unlike conventional robots, this design enables movement in any direction ‚Äî forward, backward, sideways, diagonal, and rotational ‚Äî without requiring reorientation, ideal for confined or dynamic environments.

The system integrates a Raspberry Pi with a Pi Camera for live video streaming over a Flask-based web interface. The ESP8266 microcontroller handles wireless communication and control commands, while an L298N dual H-bridge motor driver governs the speed and direction of four DC motors connected to Mecanum wheels.

A lightweight chassis supports all modules, powered by a 12V battery pack. The rover features WiFi-based control, allowing remote operation from any browser interface. It can perform complex motion such as lateral or diagonal translation and spot rotation (zero-radius turns), making it suitable for surveillance, research, and automation environments.

Future improvements include integrating computer vision for object detection, AI-based navigation, and obstacle avoidance for autonomous operation.

Tech Stack Used: Raspberry Pi, ESP8266, L298N Motor Driver, Flask (Web Interface), Python, OpenCV, Mecanum Wheels, Pi Camera, 12V Power Module.
